# 🎓 Students Performance Analytics Dashboard

A comprehensive **Students Performance Analytics Dashboard** built to analyze and visualize student academic performance, attendance, study habits, extracurricular involvement, internet access impact, and demographic insights.

---

## 📌 Project Overview

This dashboard provides meaningful insights into factors affecting student performance using interactive visualizations and filters. It helps educators, analysts, and institutions make data-driven decisions to improve academic outcomes.

The dashboard focuses on:

- Academic performance trends
- Attendance impact on scores
- Study habits vs final percentage
- Gender-based analysis
- Class-wise comparison
- Internet access influence
- Parental education impact
- Top-performing students tracking

---

# 🚀 Features

## ✅ Interactive Filters
Users can dynamically filter dashboard data using:

- Internet Access
- Gender
- Class

---

## 📈 Visual Insights Included

### 🔹 Extracurricular Activities Count
Displays participation distribution between students involved and not involved in extracurricular activities.

### 🔹 Internet Access Impact on Performance
Analyzes how internet availability affects final percentage scores.

### 🔹 Class-wise Student Count
Shows the total number of students across different classes.

### 🔹 Attendance vs Performance
Visualizes correlation between attendance percentage and academic performance.

### 🔹 Average Marks by Class
Compares average scores in:
- Math
- Science
- English

across different classes.

### 🔹 Previous Year Score vs Final Percentage
Analyzes consistency and improvement in academic performance.

### 🔹 Average Final Percentage by Gender
Compares overall academic performance between male and female students.

### 🔹 Study Hours vs Final Percentage
Shows relationship between daily study hours and final scores.

### 🔹 Parental Education vs Student Performance
Evaluates how parental education levels influence student outcomes.

### 🔹 Top 10 Performing Students
Highlights highest scoring students based on final percentage.

# 🎯 Key Insights

- Students with higher attendance generally perform better academically.
- Study hours show a positive correlation with final percentage.
- Internet access slightly improves academic outcomes.
- Parental education has measurable influence on student performance.
- Academic performance varies moderately across classes and gender.
